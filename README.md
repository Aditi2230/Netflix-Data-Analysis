# 🎬 Netflix Dashboard Analysis (Tableau)

> 📊 *Uncovering trends in Netflix content using interactive Tableau visualizations*

---

## 🚀 Project Overview

This project presents a **Tableau dashboard** analyzing Netflix's content library, focusing on movies and TV shows across different countries, genres, and ratings.

The goal is to transform raw data into **interactive insights** that help understand content distribution, audience targeting, and global trends.

---

## 📊 Dashboard Highlights

### 🔹 Key Information Displayed

* 🎥 Content Type (Movies vs TV Shows)
* 📅 Release Year & Date Added
* ⭐ Content Ratings
* 🌍 Country-wise distribution
* 🎭 Genre classification

## 📈 Key Insights

💡 **Content Distribution**

* Movies dominate Netflix’s content library compared to TV Shows

💡 **Top Countries**

* United States and India contribute the highest number of titles

💡 **Ratings Analysis**

* TV-MA and TV-14 are the most common ratings, indicating mature audience targeting

💡 **Growth Trend**

* Significant increase in content additions after 2015

💡 **Genre Popularity**

* Comedies and international content are highly prevalent
